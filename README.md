# 주식 티커 분석기

Streamlit으로 만든 간단한 **주식 티커 분석 앱**입니다. 티커를 입력하면 가격 추이, 이동평균선, 수익률, 거래량, 기업 정보를 확인할 수 ���습니다.

## 기능

- 주식 티커 입력
- 기간 선택 (1개월 ~ 5년)
- 종가 차트 표시
- 20일 / 60일 이동평균선 표시
- 기간 수익률, 최고가, 평균 거래량 요약
- 기업 기본 정보 및 소개 표시
- 최근 시세 데이터 테이블 표시

## 설치 방법

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run app.py
```

## 예시 티커

- AAPL
- TSLA
- MSFT
- NVDA
- GOOGL

## 참고

- 데이터는 `yfinance`를 통해 가져옵니다.
- 일부 티커 또는 시장 데이터는 지연되거나 제한될 수 있습니다.
